# Employee-Tracker

  ![license badge](https://img.shields.io/badge/license-MIT-brightgreen)

 
### Description
An app where you can track your employees using node, inquirer and MYsql. Mysql is used to store and organize the company data. The user is able to add roles, departments, employees, view dept and update,
